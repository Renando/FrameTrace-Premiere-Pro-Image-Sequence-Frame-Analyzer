# Security Policy / Kebijakan Keamanan

## Supported Versions / Versi yang Didukung

### 🇬🇧 English
We take the security of FrameTrace seriously. The following table describes the versions of the project that are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

### 🇮🇩 Indonesian

| Versi | Didukung |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability / Melaporkan Kerentanan

### 🇬🇧 English
If you discover a security vulnerability within FrameTrace, please **do not open a public issue**. Instead, follow these steps:
1. Please report it privately by reaching out to the maintainer directly.
2. Provide a detailed description of the issue, including steps to reproduce the vulnerability.
3. We will acknowledge your report within 48 hours and work on providing a fix or patch as quickly as possible.

We appreciate all efforts to find and disclose vulnerabilities responsibly!

### 🇮🇩 Indonesian
Jika Anda menemukan kerentanan keamanan di dalam FrameTrace, mohon **jangan membuka issue publik**. Alih-alih, ikuti langkah-langkah ini:
1. Harap laporkan secara tertutup dengan menghubungi pengelola (*maintainer*) secara langsung.
2. Berikan deskripsi mendetail tentang masalah tersebut, termasuk langkah-langkah untuk mereproduksi kerentanan.
3. Kami akan merespons laporan Anda dalam waktu 48 jam dan berupaya memberikan perbaikan atau *patch* secepat mungkin.

Kami sangat menghargai semua upaya untuk menemukan dan melaporkan kerentanan secara bertanggung jawab!

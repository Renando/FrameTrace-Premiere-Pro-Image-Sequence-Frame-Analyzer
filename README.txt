=== INSTALLATION & USER GUIDE / PANDUAN INSTALASI & PENGGUNAAN ===

🇮🇩 BAHASA INDONESIA
---------------------------------------------------
Cara Pemasangan:
1. Klik 2 kali pada file "pencet.reg", lalu klik 'Yes' ketika muncul peringatan. (Ini diperlukan untuk mengaktifkan mode debug pada ekstensi Adobe tingkat pengembang).
2. Pindahkan atau salin folder "FrameTrace" (yang berisi file index.html, css, dll.) ke lokasi direktori instalasi plugin Anda:
   Windows: C:\Program Files (x86)\Common Files\Adobe\CEP\extensions

Cara Menggunakan Plugin:
1. Buka Adobe Premiere Pro.
2. Impor image sequence (urutan gambar) ke dalam timeline Anda.
3. Potong (trim) klip sesuai kebutuhan Anda.
4. Pada menu atas, klik: Window → Extensions → FrameTrace
5. Panel FrameTrace akan muncul di layar Anda.
6. Pilih klip image sequence yang ada di timeline, dan klik tombol pada FrameTrace untuk menganalisis dan mendapatkan rentang frame aslinya.

---------------------------------------------------

🇬🇧 ENGLISH
---------------------------------------------------
Installation Guide:
1. Double click on the "pencet.reg" file, then click 'Yes' when prompted. (This is required to enable debug mode for developer-level Adobe extensions).
2. Move or copy the "FrameTrace" folder (which contains the index.html, css files, etc.) to your plugin installation directory:
   Windows: C:\Program Files (x86)\Common Files\Adobe\CEP\extensions

How to Use the Plugin:
1. Open Adobe Premiere Pro.
2. Import an image sequence into your timeline.
3. Trim the sequence clip as needed.
4. On the top menu, click: Window → Extensions → FrameTrace
5. The FrameTrace panel will appear on your screen.
6. Select the trimmed image sequence clip in the timeline, and click the analyze button in FrameTrace to calculate its original frame range.
